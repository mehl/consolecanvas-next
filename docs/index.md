# consolecanvas-next

`consolecanvas-next` is a terminal graphics library for Node.js that allows you to create and manipulate canvases in the terminal using various character-based rendering techniques. It supports multiple canvas types, including block characters and braille characters, and provides a 2D rendering context similar to the HTML5 Canvas API.
